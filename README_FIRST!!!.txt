##NOTAS DE EVALUACIÓN##

Tecnologías utilizadas:

El aplicativo ha sido desarrollado en ASP.NET MVC 4 usando el IDE Visual Studio Community 2017.
Así mismo se han utilizado las siguientes tecnologías de tereceras partes:

Json.net
Description: Framework que facilita el tratamiento de información
             de datos Json hacia el entorno .NET

License: The MIT License (MIT) Copyright (c) 2007 James Newton-King
http://www.newtonsoft.com/json

CodeRush
Description: Paquete de mejora para el IDE Visual Studio 
License: De pago, (Versión de prueba de 30 días)
https://www.devexpress.com/products/coderush/

Json2CSharp
Description: Utilidad que permite parsear datos Json a clases C#
             y viceversa.
https://marketplace.visualstudio.com/items?itemName=DangKhuong.JSONtoC

Todo el aplicativo ha sido desarrollado mediante el patron Model - Vista - Contrlador,
así mismo, para facilitar su testeo he implementado una sencilla interfaz donde
se pueden testear los diferentes métodos.
